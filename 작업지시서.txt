시스템 구축 및 유지보수 작업지시서 (Plant TI SOP)
이 섹션은 향후 시스템을 수정하거나 재구축할 때 반드시 지켜야 할 '철칙'들을 정리한 문서입니다.

1. 기본 보안 정책
관리자 마스터 비밀번호: 1207 (코드 내부에 하드코딩하며 UI상에는 절대 번호를 노출하지 않는다).

삭제 비밀번호: 각 예약 건별로 개별 비밀번호를 설정하여 본인 확인을 거친다.

2. 사용자 인터페이스 (UI/UX) 표준
폰트 가시성: 사이드바 메뉴 라벨은 일반 폰트의 300% 크기로 유지한다.

브랜드 컬러: 대우건설 Plant TI 팀의 아이덴티티를 위해 군청색(#000080)과 주황색(#FF5733) 포인트를 사용한다.

테마 지원: 다크 모드를 기본으로 지원하며, 라이트 모드 전환 시에도 모든 글자가 선명하게(고대비) 보이도록 설정한다.

안전장치: 예약 확정 버튼 위에 '최종 확인 체크박스'를 배치하며, 이 체크박스가 선택되지 않으면 예약이 절대 진행되지 않도록 로직을 고정한다. (엔터키 실수 방지 포함)

3. 글로벌 시간 동기화 로직
입력: 사용자는 해외 현지 시각(타임존 선택 포함)으로 입력한다.

검증: 입력 즉시 한국 시각(KST)으로 변환한 프리뷰 박스를 실시간 노출한다.

저장: 모든 시각은 DB에 ISO 표준(UTC)으로 저장하여 서버 시차 문제를 원천 차단한다.

목록 표시: 리스트 제목은 '해외 현지 시작 시각'을 기준으로 하되, 상세 내용에 '예약 확정 시각(KST)'을 병기한다.

4. 녹화 및 다운로드 엔진 로직
지능형 클릭: 웹 세미나 입장 시 3회 시간차(5초 간격) 클릭을 수행한다 (이메일 확인 버튼, 플레이 버튼, 입장 버튼 포함).

진단 기능: 녹화 실패 시 failure_reason 컬럼에 에러 원인을 상세히 기록하고, 이를 UI의 '녹화 결과' 섹션에 빨간색 경고창으로 노출한다.

파일 수령: 브라우저 재생 에러를 방지하기 위해 st.download_button을 이용한 노트북 직접 저장 방식을 고수한다.

5. 데이터베이스(Supabase) 필수 컬럼
title, webinar_url, email, scheduled_at, duration_min, status, video_url, is_downloaded, delete_password, failure_reason.



모든 코드 수정 시 파일 최상단에 **버전 정보(vX.X.X)**와 수정 날짜, 주요 변경 사항을 주석으로 명시할 것.

이전 버전의 핵심 기능(마스터 비번 1207, KST 시간 변환, 안전 체크박스, 노트북 직접 저장)은 삭제하지 않고 유지하며 버전업할 것.

에러 발생 시 에러 코드가 아닌 사용자가 이해할 수 있는 한글 메시지를 failure_reason에 기록할 것.
